# Project Requirements

### Functional Requirements
* **Product Catalog**: Users can view and search for school supplies.
* **Shopping Cart**: Users can add items and update quantities via Session.
* **Checkout**: Collects delivery and payment details.
* **Email Notifications**: System sends a real email receipt upon order completion.

### Technical Requirements
* **Language**: C# with ASP.NET Core 6.0/8.0.
* **Database**: Entity Framework Core with SQL Server.
* **Communication**: SMTP for real-time email delivery.
