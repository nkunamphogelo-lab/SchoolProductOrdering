# School Product Ordering System

## Project Overview
The School Product Ordering System is a web-based application developed using ASP.NET Core Razor Pages.  
The system allows learners to view and order school-related products such as laptops, stationery, and school accessories.

This project was developed as a **school assignment** to demonstrate understanding of:
- Web application development
- Database design
- Entity Framework Core
- Software documentation

---

## Features
- View school products
- Product images, prices, and descriptions
- Add products to a shopping cart
- Simple ordering process
- Admin product management (Create, Edit, Delete)

---

## Technologies Used
- Visual Studio 2022
- ASP.NET Core Razor Pages (.NET 8)
- C#
- Entity Framework Core
- SQL Server LocalDB
- Bootstrap
- HTML & CSS

---

## How to Run the Project
1. Open the solution in Visual Studio
2. Restore NuGet packages
3. Run database migrations
4. Press **Ctrl + F5** to start the application
5. Navigate to `/Products` to view products
